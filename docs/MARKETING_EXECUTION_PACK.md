# Marketing execution pack

## Core offer

Free 60-day Growth beta for Shopify merchants or agencies with physical catalogs, supplier price lists, or frequent bulk product edits.

## Lead promise

"Find which SKUs are below target margin, how much inventory money is at risk, and what happens if supplier costs rise."

## Prospect filters

Prioritize:
- Shopify stores with 100+ visible products.
- Apparel, accessories, parts, home goods, beauty, supplements, hobby supplies, pet products.
- Stores mentioning wholesale, distributors, import, suppliers, catalog, or stockists.
- Agencies that manage Shopify catalog operations.

Avoid:
- digital-only products;
- tiny single-product stores;
- stores already dominated by subscriptions/services;
- regulated categories where claims/support burden is high.

## Call script

1. Ask how supplier costs are updated today.
2. Ask whether product costs live in Shopify, CSV, ERP, or spreadsheets.
3. Run scan.
4. Show highest inventory risk.
5. Run supplier cost +8% what-if.
6. Export findings.
7. Ask: "Would this have changed what you checked before a promo, reorder, or supplier update?"
