# Paid ads test plan

Do not start paid ads until:
- The app is full visibility.
- Billing and onboarding are stable.
- At least 3 merchants have installed from direct outreach.
- The first scan flow is clearly understood.

## Shopify App Store ads
Start with a tiny test only after the above is true.

Budget:
- $5-$15/day for 7 days.

Keyword themes:
- margin
- profit margin
- product costs
- pricing
- inventory
- analytics

Success threshold:
- Pause if there are no installs after 100+ qualified clicks.
- Continue only if install cost looks reasonable against $15/month pricing.

## Retargeting
Wait until there is meaningful traffic and a clean landing domain. Do not set up retargeting before the site has a real domain and privacy wording is reviewed.
